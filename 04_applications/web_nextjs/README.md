# DOMAURA App Web MVP

Application principale pour livrer la premiere demo DOMAURA.

## Lancer en local

```bash
npm install
npm run dev
```

URL par defaut: `http://localhost:3000`

## Parcours de demo

1. Home: comprendre la promesse DOMAURA
2. Explore: parcourir immobilier, produits et services
3. Listing detail: verifier vendeur, prix et score de confiance
4. Create: simuler une publication
5. Chat: simuler la conversation acheteur/vendeur
6. Checkout: simuler l'escrow mock
7. Profile: voir les badges confiance

## Important

Cette version n'a pas encore de backend reel. Les donnees sont mockees dans `data/listings.ts`.
