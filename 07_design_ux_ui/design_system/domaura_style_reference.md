# Domaura Style Reference (Web + Mobile)

This file maps the provided visual reference to implementation tokens.

## Core palette
- Navy: `#0F172A`
- Gold: `#D4AF37`
- White: `#F8FAFC`
- Muted text: `#94A3B8`
- Borders: `#E2E8F0`

## Typography
- Title: `Montserrat` (or nearest fallback)
- Body: `Inter` (or nearest fallback)

## UI language
- Rounded corners: `12px` to `20px`
- Soft shadow depth: low to medium
- Spacious cards and clean spacing
- Bottom navigation on mobile
- Listing cards + detail panel + profile + add-property flows

## Feature parity baseline
- Home
- Listings with filters
- Property detail CTA (`Contact / Book`)
- Profile
- Add property
