## 🚀 Development Workflow

### **Making a Feature**

```bash
# 1. Create branch
git checkout -b feature/my-feature

# 2. Make changes
# 3. Test locally
make test:all

# 4. Format code
make lint:fix

# 5. Commit (conventional commits)
git commit -m "feat(module): add awesome feature"

# 6. Push & create PR
git push origin feature/my-feature
# Open PR → Code review → Merge
```

### **Daily Development**

```bash
# Morning: Start services
make docker:up
make dev:all

# During day: Code, test, commit
make test:watch
make lint:fix

# Evening: Push changes
git push origin feature/branch

# Next morning: Sync with team
git rebase origin/develop
```
